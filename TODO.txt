ID do usuário é usado para validar acesso a contas e transações. Cada conta e transação tem uma coluna com o id do usuário. Este número é acessado pelo variável $_SESSION['id_usuario'].

Separar SQL de visão

Para executar SQL, é necessário chamar uma função. Essa função devolve um array de objetos. Esse resultado é usado para gerar a visão.

Com essa separação de modelo e visão, é possível gerar múltiplas visões para desktop e celulares.

Instalar tabelas

Usar prefixo "plancont_"

Remover tabelas

Remover as quatro tabelas "plancont_*"

Tabelas SQL

PK = primary key
F  = foreign key

* plancont_usuario
    PK id
	PK nome
	   email
	   sal
	   senha

* plancont_tipo_de_conta
    PK id
	   nome
	   sinal
	   
* plancont_conta
    PK id
	F  id_usuario
	F  id_tipo_de_conta
	   nome

* plancont_transacao
	PK id
	F  id_usuario
	   valor
	   data_criada
	F  debito
	F  credito
	   descricao
